# SSE LLM

## Run a test locally

On Linux / MacOS:
```console
export api_key=your token
./mvnw gatling:test
```

On Windows:
```console
set api_key=your token
mvnw.cmd gatling:test
```

## Run the Gatling Recorder

On Linux / MacOS:
```console
./mvnw gatling:recorder
```

On Windows:
```console
mvnw.cmd gatling:recorder
```
